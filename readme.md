# Задание 4. Парольный взломщик

## Ситуация  
У тебя есть zip-файл, запароленный 4-значным PIN-кодом (0000–9999). Нужно подобрать пароль перебором.

---

## Что нужно сделать  

### 1. Функция проверки  
Напиши функцию `try_password(password)`, которая:

- Проверяет, подходит ли пароль (для теста пусть правильным будет `"1234"`).  
- Если пароль верный — печатает `Найден пароль: 1234` и возвращает `True`.

---

### 2. Однопоточный перебор  

- Перебери **все пароли от 0000 до 9999 в одном потоке**.  
- Замерь время выполнения.

---

### 3. Многопоточный перебор  

Перебери те же пароли, разбив диапазон на 4 части:

- Поток 1: `0000–2499`  
- Поток 2: `2500–4999`  
- Поток 3: `5000–7499`  
- Поток 4: `7500–9999`  

---

### 4. Остановка потоков  

- Как только один поток нашёл пароль — останови все остальные.  
- Используй `threading.Event()` для синхронизации.

---

## Ожидаемый результат  

- Перебор в 4 потока работает быстрее.  
- После нахождения пароля программа не продолжает перебор.
